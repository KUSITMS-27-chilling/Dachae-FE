### ✨ 서비스 소개
**_당신의 인생을 다시한번, 앵콜_**
> '앵콜'은 인생의 이모작을 준비하는 시니어 세대들에게 재취업/창업에 대한 정보를 모아주고, 일방적인 정보제공에서 벗어나 쌍방향 소통을 지원하는 커뮤니티 서비스 입니다.
